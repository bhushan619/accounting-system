# Payroll Leave Types Feature

## Summary
Added leave type tracking to the attendance system with the following categories:
- Sick Leave (paid)
- Casual Leave (paid)
- Annual Leave (paid)
- Unpaid Leave (deducted from salary)
- Other Leave
- Leave Notes (optional comments)

## Key Points
- Only **Unpaid Leave** causes salary deductions
- All leave types support half-days (e.g., 0.5)
- Excel template includes all leave type columns
- Validation warns if total leave doesn't match absent days
- Leave breakdown displayed in attendance history table
